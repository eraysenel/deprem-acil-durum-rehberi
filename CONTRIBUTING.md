# Katkı Rehberi 💚

**İyilik İçin Yapay Zeka** topluluğuna hoş geldin! Bu proje gönüllülerle büyüyor ve **her seviyeden** katkıyı memnuniyetle karşılıyoruz. İlk defa açık kaynağa katkı verecek olsan bile.

## İlk defa mı katkı veriyorsun?
Hiç sorun değil, adım adım gidelim:

### 1. Bir görev seç
[`good first issue`](../../issues?q=is%3Aopen+label%3A%22good+first+issue%22) etiketli issue'lar yeni başlayanlar için seçilmiştir. Beğendiğin birine yorum yaz: **"Bunu üstlenebilir miyim?"** - sana atayalım, böylece iki kişi aynı işi yapmaz.

### 2. Projeyi çalıştır
Kurulum yok. Bu proje tek bir `index.html` dosyası:
- Repoyu **fork**'la, bilgisayarına indir: `git clone <senin-fork-adresin>`
- `index.html`'i çift tıklayıp tarayıcıda aç. Hepsi bu.
- Değiştir → kaydet → tarayıcıyı yenile → sonucu gör.

### 3. Değişikliğini yap
- Yeni bir dal aç: `git checkout -b ozellik-adi`
- Kodu sade ve okunur tut. Arayüz metinleri **Türkçe**.
- Erişilebilirliği gözet: iyi kontrast, büyük dokunma alanları, okunur yazı.

### 4. Pull Request (PR) aç
- `git push` → GitHub sana "Compare & pull request" çıkaracak.
- Açıklamada hangi issue'yu çözdüğünü yaz: **"Closes #12"**.
- Mümkünse bir ekran görüntüsü ekle - inceleme hızlanır.

## İlkelerimiz
- **Sadelik:** bağımlılık eklemeden, tek dosyada kalmaya çalışıyoruz.
- **Erişilebilirlik:** herkes kullanabilmeli (yaşlı, engelli, düşük bağlantı).
- **Ücretsiz & açık:** hiçbir şey para istemez, hiçbir veri satılmaz.
- **Saygı:** nazik ol, yardımlaş. Kötü davranışa yer yok.
